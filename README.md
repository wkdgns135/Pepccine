## 📌 Introduction
좀비 아포칼립스 속, 주인공은 백신을 찾아 폐허가 된 건물에 들어간다.
그러나 손에 넣은 것은 백신이 아닌 정체불명의 음료수.
좀비와의 치열한 전투 속에서 생존을 위해 음료수를 마셨지만,
그 정체는 과연 무엇일까?
플레이어의 선택과 생존 방식에 따라 결말이 달라지는
하드코어 액션 슈팅 로그라이크!

## 🎮 Features
- **로그라이크 + 슈팅 결합**: 맵 랜덤 생성 & 웨이브 기반 전투
- **아이템 조합 시스템**
- **고유한 좀비 컨셉**
- **UE5 기반의 고퀄리티 그래픽**
